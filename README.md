# WordPress Contributions Portfolio

Personal portfolio of WordPress work, learning materials, and supporting assets.

## Repository structure
- `projects/` – client or personal builds. Each project includes context, highlights, and follow-up tasks.
  - `personal-portfolio-theme/` – block-based theme example with changelog and TODO list.
- `learning/` – study notes, research, and checklists such as block editor tips and performance review template.
- `assets/` – shared media (`screenshots/`, `design/`, `exports/`) plus guidance for managing files.

> Tip: Keep every project self-contained with its own README, CHANGELOG, and TODO so you can show progress over time.

## Getting started
1. Clone the repo: `git clone https://github.com/zwanski2019/wordpress-contributions.git`.
2. Browse `projects/` to review delivered work and outstanding tasks.
3. Open `learning/` when preparing for new features or audits.
4. Add screenshots or exports to `assets/` and link them from the project docs.
