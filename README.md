# WordPress Contributions Portfolio

This repository collects personal contributions, study notes, and resources related to WordPress projects.

## Structure
- `projects/` – snapshots, notes, or code snippets from custom themes, plugins, or site builds.
- `learning/` – research notes, tutorials, and references gathered while working on WordPress tasks.
- `assets/` – supporting screenshots and design assets.

> Tip: Organize each contribution in its own subfolder with a short README so you can describe the goal, the solution, and any lessons learned.
